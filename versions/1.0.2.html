Changes

• Made various changes to the config options and categories.

• When chopped, logs drop 1 to 3 Campfire Logs (the amount is random), instead of dropping only 1.

Additions

• Added compatibility with Oh The Biomes You'll Go.
    - You can build two new Campfires, Boric Campfire, which requires Brim Powder and Cryptic Campfire, which requires Cryptic Magma Block.
    - Those Campfires also have their own config options similar to Vanilla Campfire's configs.
